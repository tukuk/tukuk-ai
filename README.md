# Tukuk AI

Tukuk AI ialah satu website pelajar percuma yang dibina sebagai aplikasi statik untuk GitHub Pages.

## Ciri utama

- AI chat client-side
- Music engine menggunakan Web Audio API
- Video visualizer menggunakan Canvas
- Image prompt/artboard generator
- Student dashboard: pelan belajar, kuiz, flashcards, mode fokus, portfolio
- Live app builder dan code editor
- Multilingual prompt detection
- Theme Light/Dark
- Export chat, template, screenshot, dan semua output
- Senarai alat percuma/open-source

## Cara guna

Buka website:

```text
https://tukuk.github.io/tukuk-ai/
```

Edit fail utama:

```text
index.html
```

## Deploy

Repo ini menggunakan GitHub Pages dari branch `main`.

```bash
git add index.html README.md .gitignore
git commit -m "Update Tukuk AI"
git push
```
